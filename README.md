# **Hold 2 Holds** 
Hold 2 Holds is an online store dedicated to climbing enthusiasts.
The website was created as part of my frontend learning journey — a practice in building a clean, modern, and responsive e-commerce layout using HTML, CSS, and basic JavaScript.

### Features 
- Recommended product catalog with climbing holds, volumes, footholds, bolts, and T-nuts.
- Product descriptions with pricing and “Add to cart” functionality.
- Responsive design for desktop and mobile users.
- About Us section highlighting the story and mission of Hold 2 Holds.
- Team profiles showcasing the founders and their climbing expertise.
- Basic interactivity implemented with JavaScript (e.g. simple UI effects).

### Technologies
- **HTML5** – page structure and content organization  
- **CSS3** – styling, layout and responsive design 
- **JavaScript** – small interactive features
